# dataStructure and algorithm     数据结构和算法

## 数据结构

### 逻辑结构

* 集合结构
    > 相互之间没有关系

* 线性结构
    > 一对一的关系

* 树形结构
    > 一对多的关系

* 图形结构
    > 多对多的关系
    
### 物理结构

* 顺序存储结构
    > 数据存储在地址连续的存储单位里，数据间的逻辑关系和物理关系是一致的

* 链式存储结构
    > 数据可以存放在任意位置，并有一个指针存放数据元素的地址
